# Superstore Sales Analysis (Excel)

This project analyzes Superstore sales data using Microsoft Excel.
The goal was to evaluate sales performance, profitability, and seasonal trends
across regions and product categories.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Dashboard design

## Key KPIs
- Total Sales: $2.67M
- Total Profit: $145,976.94
- Profit Margin: ~11–12%
- Total Customers: 10,952

## Key Insights
- Technology is the top-performing category, contributing ~32% of total sales.
- Sales peak toward the end of the year, showing strong seasonality.
- Profit varies significantly by region.
- Shipping cost outliers impact overall profitability.
